Para ejecutar Practica3: java src.edd.Encerrado

Para la realización de este proyecto hicimos la clase Arboljuego para formar el arbol de minimax
con enteros. Los métodos del juego se encuantran en la clase Encerrado.
Al abrir el juego aparece un menú en donde se puede escoger la opción de leer instrucciones, jugar
o salir.
Para cambiar el modo de juego durante la partida solamente se debe ingresar el número -2.

En este proyecto discutimos bastante cuál sería un buen método para implementar minimax y decidimos
ir formando el árbol binario mientras utilizabamos backtracking.

Los únicos problemas que se presentaron fueron básicamente que al principio no entendíamos muy bien cómo
funcionaba el minimax y cómo formaríamos el árbol. También nos tardamos un poco al momento de hacer el 
caso base para darle valores a las hojas.
